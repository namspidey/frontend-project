# Social Network Project - Frontend
A modern Instagram-like frontend built with React.js . It connects to a Node.js + MongoDB backend to provide social media features like login, photo upload, feed, chat, and profiles.

# Features:

- Authentication:
Register / Login using backend API

- Home Feed:
View posts from followed users
Like / Unlike posts in real-time
Comment (optional)

- Profiles:
View your own profile
Visit other users' profiles
Follow / Unfollow users

- Photo Upload:
Upload new posts with caption
Preview before uploading

- Chat (Realtime):
1-on-1 chat using Socket.io
Supports both text and image messages

- Search:
Search users by username

- Responsive Design:
Works perfectly on desktop and mobile
